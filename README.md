**Banking API - Spring Boot
Description**

**REST API developed in Java using Spring Boot to simulate core banking operations such as account creation, deposits, withdrawals, and transfers.**

**Features**
Create account,
List all accounts,
Get account by ID,
Deposit funds,
Withdraw funds,
Transfer between accounts.


**Technologies Used**
Java,
Spring Boot,
Spring Web,
Spring Data JPA,
H2 Database,
Maven,
Swagger (OpenAPI),
HATEOAS.


**Project Structure**
controller,
service,
repository,
model,
dto,
exception.


**Validation and Error Handling**
Input validation using @Valid,
Global exception handling with @RestControllerAdvice,
Standardized error responses


**How to Run the Project**
mvn spring-boot:run.
